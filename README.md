# My Blog @ Docusaurus 2

This directory is a blog built with [Docusaurus](https://v2.docusaurus.io) site that can be deployed to Vercel with zero configuration.

## Deploy Your Own

Deploy your own Docusaurus project with Vercel.

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/vercel/vercel/tree/main/docusaurus-2&template=docusaurus-2)

_Live site: https://blog.kitkong.xyz

### How We Created This Example

To get started with Docusaurus on Vercel, you can use the [Docusaurus CLI](https://v2.docusaurus.io/docs/installation#scaffold-project-website) to initialize the project:

```shell
npx create-docusaurus@latest my-website classic
```
